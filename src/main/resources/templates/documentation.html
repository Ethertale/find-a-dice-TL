Features

Events

Users can create events and mark if they are interested or going.

Groups

Users can create groups and gather together in a place where they can communicate.

Guides

Races, Classes, Alignments, Backgrounds, you name it.

Chats

Users can begin a chat that creates a chat room between two where users can communicate privately.

Dice Rolling System

The site offers a dice-rolling system that players can use if they don't have dice on hand. d4, d6, d8, d10, d12, d20, d%
